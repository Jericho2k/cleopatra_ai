# Cleopatra AI 🐍

> Built for OFM agencies that refuse to sound like a bot.

---

## What it does

Cleopatra sits behind your chatter dashboard and suggests 3 AI-generated replies for every incoming fan message — trained on the creator's real voice, adapting to conversation stage in real time.

## Features

- 🎭 **Persona engine** — learns each creator's exact voice, vocabulary, and emoji habits
- 🧠 **Stage awareness** — detects where every conversation is and responds accordingly  
- 💰 **Fan scoring** — identifies whales and high-value fans automatically
- ⚡ **Real-time suggestions** — 3 reply options per message, instantly

## Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python 3.11 + FastAPI |
| Database | Supabase (Postgres + pgvector) |
| AI | Together AI — Llama 3.1 70B |
| Frontend | Next.js |
